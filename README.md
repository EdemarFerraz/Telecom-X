1. Objetivo
Analisar a base de dados da Telecom X para identificar padrões que levam ao cancelamento de clientes (Churn) e propor ações estratégicas de retenção.

2. Processamento dos Dados
Limpeza: Conversão de cobranças para valores numéricos e remoção de duplicados.

Enriquecimento: Criação da métrica Contas_Diarias para avaliar o custo diário do serviço.

Padronização: Tradução de variáveis e binarização de categorias (Ex: Sim/Não para 1/0).

3. Principais Descobertas (Insights)
Tipo de Contrato: O contrato Mensal é o maior gerador de evasão. Clientes sem fidelidade anual têm uma barreira de saída muito baixa.

Impacto Financeiro: Clientes com mensalidades mais altas tendem a cancelar mais, sugerindo sensibilidade ao preço ou busca por ofertas da concorrência.

Fator Tempo (Tenure): O risco de churn concentra-se nos primeiros 6 meses. Clientes veteranos apresentam alta fidelidade.
